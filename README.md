# 📊 Student Report Generator

A simple Command Line Interface (CLI) application built with **Dart** that generates student reports based on their academic information. This project was created to practice Dart programming and Object-Oriented Programming (OOP) concepts through a real-world mini project.

---

## 🚀 Features

- Add student information
- Store student records
- Generate student reports
- Display report details in the terminal
- Menu-driven CLI interface
- Beginner-friendly code structure

---

## 🛠️ Built With

- Dart
- Command Line Interface (CLI)

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/sharifahmedd/student-report-generator.git
```

### 2. Navigate to the project folder

```bash
cd student-report-generator
```

### 3. Run the application

```bash
dart run assignment.dart
```
